ABOUT MODULE
------------

This module is used to get login via REST API console Salesforce to Drupal 7 sites.

There is only one person who contributed to build up this API module to access Drupal site through Salesforce.

With the help of this module user can easily login into Drupal site without having registration step in it. Accessing Salesforce API's need to have salesforce button to act on API for it.


INSTALLATION
------------

1) Download the module file (compressed file) from the Drupal.org.

2) Unzip it into /sites/all/modules/ or /sites/all/modules/contrib/.

3) Login to Drupal admin (back-end) panel.

4) Go to -> Modules -> Salesforce(locate) -> Salesforce SSO and enable it.

5) After enabling, go to configuration of module and set configuration and SAVE settings.

6) Now your module is ready to go. Please test it with Salesforce account and try to login directly on site.

7) You can check the users in "People" tab, user will be created/registered with Salesforce credentials.
